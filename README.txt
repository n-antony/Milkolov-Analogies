The zip file contains a python file "ngupta16.py", word-text.txt, problem3.txt and Assignemnt 5 report.

Before extrcting the zip file please make sure you have installed the gensim package:
1. Open command prompt
2. Type "pip install --upgrage gensim" to install the gensim package


Now extract the zip file ngupta16.
1. Please make sure you download and extract the pre trained embedding in ngupta16 folder before executing the program.
a). 'glove.twitter.27B.50d.txt' can be downloaded from http://nlp.stanford.edu/data/glove.twitter.27B.zip
b) 'glove.6B.100d.txt' can be downloaded from http://nlp.stanford.edu/data/glove.6B.zip
2. Once you have both the .txt files in ngupta16 folder go to command prompt.
3.Type "py ngupta16.py" to execute the file and get the output

Note: 
1. Folder contains "ngupta16.py" that has accuracy of the 8 analogies mentioned the analogy set, antonyms of the given word and accucary of my own analogy datasets
